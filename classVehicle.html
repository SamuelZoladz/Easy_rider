Detailed Description

Base vehicle with longitudinal dynamics (IDM) and routing policy.

Movement model:

  • Position is tracked along the current edge as a scalar [0 -> length].
  • Speed is integrated using IDM; the free-flow target speed is IDMParams::v0.
  • Effective speed on an edge is limited by the congestion model.
  • On reaching the end of an edge, the next edge from route_ is taken.

Rerouting:

  • When congestion is detected (e.g., on edge entry), the vehicle may recompute its route using the configured strategy after a cooldown.
