Detailed Description

Base vehicle with simple longitudinal dynamics and routing policy.

Movement model:

  • position is tracked along the current edge as a scalar [0 -> length].
  • speed integrates with acceleration/braking toward a target speed limited by both the vehicle's maxSpeed and the edge's effective speed reported by CongestionModel.
  • On reaching the end of an edge, the next edge from route_ is taken.

Rerouting:

  • When congestion threshold is detected (e.g., on edge entry), the vehicle may recompute its route using the configured strategy.
